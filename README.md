# 🔤 Mapeamento de Caracteres no Alfabeto em C

Programa desenvolvido em **Linguagem C** que recebe uma frase ou palavra e exibe a posição correspondente de cada letra no alfabeto (onde `a = 1`, `b = 2`, `z = 26`).

---

## 💻 Sobre o Projeto

O objetivo principal do projeto é processar uma entrada de texto do usuário, filtrar os caracteres alfabéticos minúsculos e utilizar **vetores de frequência** e a **tabela ASCII** para mapear e exibir o índice de cada caractere presente na palavra.

### 🛠️ Funcionalidades
- Leitura de strings longas via `fgets`.
- Mapeamento direto de letras para sua posição alfabética (1 a 26).
- Filtragem automática de caracteres não alfabéticos.
